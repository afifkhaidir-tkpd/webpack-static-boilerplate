Types

There are four types of elements in form component, vanilla, disabled, big text field, and dropdown.

Guidelines

Base form elements includes text box, icons, the form length varies on usage, user can put 24px icon on either end of the box, or no icons depending on usage.

Padding

(1) Text box with full layout (2 icons on both ends), (2), Use character or word counters where there is a character or word limit, text is right justified. (3) Big text field height is fixed at 150px
